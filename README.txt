Платформа записи эфира радиопотоков.

Библиотеки

manager - Постановка и управление задачами для воркеров, сохранения результата/ошибок.
worker - Выполнение задач и отправка результатов на трекер.

Команды

tracker - RPC-сервер. Обертка логики менеджера.
ripper - Продвинутый HTTP-клиент. Записывает эфир по задачам от воркера.
